# 🐍 Snake Game

A simple yet addictive Snake Game built using **HTML, CSS, and JavaScript**. Guide the snake, eat the food, and grow as long as possible without crashing into walls or yourself.

---

## 🎮 Features

* Smooth snake movement
* Real-time score tracking
* Game over detection
* Keyboard controls (Arrow keys)
* Responsive and minimal UI

## 🛠️ Tech Stack

* **HTML** → Structure of the game
* **CSS** → Styling and layout
* **JavaScript** → Game logic and interactions

## 🎯 How to Play

* Use **Arrow Keys** to control the snake
* Eat the food to grow longer 🍎
* Avoid hitting:

  * Walls ❌
  * Yourself ❌

Game ends when you collide. Try to beat your high score!
